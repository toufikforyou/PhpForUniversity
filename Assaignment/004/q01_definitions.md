# Question 1: Definitions

## CSS Grid Layout
CSS Grid Layout is a two-dimensional layout system for the web. It lets you lay out items in rows and columns. It has the ability to control the sizing and positioning of boxes and their contents.

## CSS Grid Elements
These are the direct children of a grid container. When you apply `display: grid` to a parent element, all its direct children automatically become grid items (elements).

## CSS Grid
Refers to the entire grid structure created by the intersection of horizontal and vertical lines â€“ one set defining columns and the other rows. Elements can be placed onto the grid within these column and row lines.

## CSS Grid Container
The element on which `display: grid` or `display: inline-grid` is applied. It is the parent of all the grid items.

## CSS Grid Item
The direct children of the grid container.

## Responsive Web Design
Responsive Web Design is an approach to web design that makes web pages render well on a variety of devices and window or screen sizes. It uses flexible layouts, images, and CSS media queries.

## Media Query
Media queries are a feature of CSS3 allowing content rendering to adapt to different conditions such as screen resolution (e.g., smartphone screen vs. computer screen). It uses the `@media` rule to include a block of CSS properties only if a certain condition is true.
